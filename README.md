# Christmas miniature

A small Christmas miniature developed with
* Three.JS
* Blender
* Webpack

![Christmas miniature](christimas-mininature-mov.gif)


## Run the application
Make sure to install first all the needed dependencies.
```shell
    npm install
```

Then to run the application type: 
```shell
    npm start
```

## Build the application
 
```shell
    npm run build
```
